# Phase 1 Playwright Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a runnable TypeScript/Playwright foundation with quality checks, a versioned UI-exercise inventory, isolated test data, the first Notes API lifecycle test, the first Notes UI login test, and Chromium CI.

**Architecture:** Playwright specifications depend on focused Page Objects and typed API clients. Test-data factories create unique accounts, while tests own cleanup. A generated inventory captures the platform surface before later implementation phases classify and automate every compatible exercise.

**Tech Stack:** Node.js 24, TypeScript, Playwright Test, ESLint flat config, Prettier, and GitHub Actions on Windows development hosts and Ubuntu CI runners.

## Global Constraints

- Use TypeScript and Playwright Test.
- Use `npm.cmd` and `npx.cmd` in PowerShell because local script execution policy blocks `npm.ps1` and `npx.ps1`.
- UI base URL is `https://practice.expandtesting.com`.
- Notes API base URL is `https://practice.expandtesting.com/notes/api`.
- Chromium is the blocking browser in Phase 1; Firefox and WebKit enter the full regression in a later phase.
- Mutable tests use unique generated users and delete them in `finally` blocks.
- Never commit secrets, `.env` files, browser binaries, reports, traces, screenshots, videos, `.firecrawl/`, or `.superpowers/`.
- The owner performs each checkbox step in Visual Studio Code, verifies the result, and commits only after the task passes.

```ts
export interface EnvironmentConfig {
  webBaseUrl: string;
  notesApiBaseUrl: string;
}

function normalizedHttpUrl(value: string, variableName: string): string {
  const url = new URL(value);
  if (!['http:', 'https:'].includes(url.protocol)) {
    throw new Error(`${variableName} must use http or https`);
  }
  return url.toString().replace(/\/$/, '');
}

export function loadEnvironment(source: NodeJS.ProcessEnv = process.env): EnvironmentConfig {
  return {
    webBaseUrl: normalizedHttpUrl(
      source.WEB_BASE_URL ?? 'https://practice.expandtesting.com',
      'WEB_BASE_URL',
    ),
    notesApiBaseUrl: normalizedHttpUrl(
      source.NOTES_API_BASE_URL ?? 'https://practice.expandtesting.com/notes/api',
      'NOTES_API_BASE_URL',
    ),
  };
}
```

```ts
export type ExerciseCompatibility = 'review-required' | 'playwright' | 'excluded';
export type ExerciseStatus = 'planned' | 'implemented' | 'excluded';

export interface RawExercise {
  name: string;
  url: string;
}

export interface UiExercise extends RawExercise {
  category: string;
  compatibility: ExerciseCompatibility;
  status: ExerciseStatus;
  reason?: string;
  testFile?: string;
}

export function normalizeExercises(raw: RawExercise[]): UiExercise[] {
  const unique = new Map<string, RawExercise>();

  for (const item of raw) {
    const name = item.name.trim();
    if (name && !unique.has(item.url)) unique.set(item.url, { name, url: item.url });
  }

  return [...unique.values()]
    .sort((left, right) => left.name.localeCompare(right.name))
    .map((item) => ({
      ...item,
      category: 'unclassified',
      compatibility: 'review-required',
      status: 'planned',
    }));
}
```

- [ ] **Step 4: Create `scripts/build-ui-inventory.ts`**

```ts
import { writeFile } from 'node:fs/promises';
import { chromium } from '@playwright/test';
import { normalizeExercises, type RawExercise } from '../inventory/ui-exercise';
import { loadEnvironment } from '../utils/environment';

const environment = loadEnvironment();

async function main(): Promise<void> {
  const browser = await chromium.launch();

  try {
    const page = await browser.newPage();
    await page.goto(environment.webBaseUrl);

    const raw = await page.locator('h3 a').evaluateAll((links) =>
      links.map((link) => ({
        name: link.textContent ?? '',
        url: (link as HTMLAnchorElement).href,
      })),
    );

    const exercises = normalizeExercises(raw as RawExercise[]);
    if (exercises.length < 40) {
      throw new Error(`Expected at least 40 exercise links, found ${exercises.length}`);
    }

    await writeFile('inventory/ui-exercises.json', `${JSON.stringify(exercises, null, 2)}\n`);
    console.log(`Wrote ${exercises.length} exercises to inventory/ui-exercises.json`);
  } finally {
    await browser.close();
  }
}

void main();
```

- [ ] **Step 4: Create `api/users.client.ts`**

```ts
import type { APIRequestContext, APIResponse } from '@playwright/test';
import type { TestUser } from '../data/user.factory';
import type {
  ActionResponse,
  ApiEnvelope,
  ApiResult,
  AuthenticatedUser,
  RegisteredUser,
} from './models';

export class UsersClient {
  constructor(
    private readonly request: APIRequestContext,
    private readonly baseUrl: string,
  ) {}

  async register(user: TestUser): Promise<ApiResult<ApiEnvelope<RegisteredUser>>> {
    const response = await this.request.post(`${this.baseUrl}/users/register`, {
      form: user,
    });
    return this.toResult<ApiEnvelope<RegisteredUser>>(response);
  }

  async login(email: string, password: string): Promise<ApiResult<ApiEnvelope<AuthenticatedUser>>> {
    const response = await this.request.post(`${this.baseUrl}/users/login`, {
      form: { email, password },
    });
    return this.toResult<ApiEnvelope<AuthenticatedUser>>(response);
  }

  async deleteAccount(token: string): Promise<ApiResult<ActionResponse>> {
    const response = await this.request.delete(`${this.baseUrl}/users/delete-account`, {
      headers: { 'x-auth-token': token },
    });
    return this.toResult<ActionResponse>(response);
  }

  private async toResult<T>(response: APIResponse): Promise<ApiResult<T>> {
    return {
      httpStatus: response.status(),
      body: (await response.json()) as T,
    };
  }
}
```
